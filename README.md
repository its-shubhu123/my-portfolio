# 💼 Shubham Vishwakarma — Developer Portfolio

A modern, responsive **developer portfolio website** built using **React (Vite) + Tailwind CSS** to showcase my projects, skills, and experience.

---

## 🚀 Live Demo

👉 https://your-portfolio.vercel.app

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🎞️ Framer Motion (animations)
* 🌙 Dark/Light Mode
* 📦 Component-based architecture

---

## ✨ Features

* Responsive design (mobile + desktop)
* Modern UI with smooth animations
* Dark/Light mode toggle
* Project showcase with GitHub links
* Skills and experience sections
* Contact section with social links
* Clean and reusable component structure

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/ts-shubhu123/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
